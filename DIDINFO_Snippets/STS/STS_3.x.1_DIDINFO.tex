\DIDINFO{STS-3.x.1 :: This subsection shall describe the procedures necessary prepare the hardware item(s) under test and and related hardware for the test. 
Reference may be made to published operating manuals for these procedures. 
The following information shall be provided, as applicable and as may be necessary:
\begin{enumerate}[itemindent=5pt,topsep=0pt,itemsep=0pt,partopsep=0pt, parsep=0pt, label=(\alph*)]
\item Specific hardware to be used, identified by name and, if applicable, model, serial, and/or version number,
\item Any switch settings, configuration parameters, and/or cabling necessary to connect the hardware,
\item Diagrams to show hardware, interconnecting control, and data paths, and
\item Step-by-step instructions for placing the hardware in a state of readiness.
\end{enumerate}}
