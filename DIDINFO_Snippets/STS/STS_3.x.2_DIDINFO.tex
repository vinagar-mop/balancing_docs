\DIDINFO{STS-3.x.2 :: This subsection shall describe the procedures necessary to prepare the software item(s) under test and any related software, including data, for the test. 
Reference may be made to published software manuals for these procedures. 
The following information shall be provided, as applicable and as may be necessary:
\begin{enumerate}[itemindent=5pt,topsep=0pt,itemsep=0pt,partopsep=0pt, parsep=0pt, label=(\alph*)]
\item The specific software to be used in the test, identified by name and, if applicable, version number,
\item The storage medium and archival plans for data collected from the item(s) under test,
\item The storage medium and archival plans of any related software (e.g., simulators, test drivers, databases),
\item Instructions for loading the software, including required sequence, and
\item Instructions for software initialization common to more than one test case.
\end{enumerate}}
